# firewall-linux
Mini Linux Firewall

Projet d'étude sur le développement de modules noyau Linux (Kernel Modules).

Ce projet a pour objectif d'implémenter un pare-feu simple au niveau du noyau en utilisant le framework Netfilter. Il m'a permis d'approfondir mes connaissances sur la gestion des paquets réseau, la programmation système en C et l'interaction avec le kernel Linux.
Origine du projet

Ce projet est basé sur les travaux initiaux de Vivek Shankar. Il a été choisi comme base pour des exercices d'apprentissage et a fait l'objet de modifications et de refactorisation pour une meilleure compréhension des mécanismes internes.
